# 使用新浪股票API端口爬取实时股票数据

### 1. 新浪股票数据接口
-----
#### 深股 http://hq.sinajs.cn/list=sz0***** 

#### 新股 http://hq.sinajs.cn/list=sz3*****

#### 沪股 http://hq.sinajs.cn/list=sh6******
